On January/15th/2021 Synthetics will be adding new Ip addresses for several Synthetics Locations.
To ensure your monitors are not effected by these changes please add the appropriate Ip addresses to your firewalls allow List.


For more information about Synthetics Minion Ip/DNS addresses visit [Synthetic monitor public minion IPs](/docs/synthetics/synthetic-public-minion-ips)
For the full list of Ip/DNS addresses that will be active on January/15/2021 visit [Daily JSON listings for IP addresses](/docs/synthetics/synthetic-public-minion-ips/#dynamic-ip-addresses)
