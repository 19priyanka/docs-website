As of [December 2, 2020](https://discuss.newrelic.com/t/first-notice-limiting-access-to-new-relic-plugins/121832), New Relic Plugin access has been limited to accounts that have accessed a legacy Plugin in the 30 days before that. The legacy New Relic Plugin experience reached end of life (EoL) June 16, 2021. For more information, see our [Explorers Hub post](https://discuss.newrelic.com/t/new-relic-plugin-eol-wednesday-june-16th-2021/127267).

For related New Relic offerings, see:

* **[newrelic.com/integrations](https://newrelic.com/integrations):** See all the technologies and services New Relic provides integrations for. 
* **[developer.newrelic.com](https://developer.newrelic.com/):** Use developer tools to collect data from any source, automate workflows, build apps, and use our APIs.
