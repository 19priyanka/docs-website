New Relic previously offered a private beta to collect performance data from HTML pages formatted for [Google's Accelerated Mobile Pages (AMP) project](https://www.ampproject.org/learn/overview/). The private beta is closed.

<Callout variant="important">
  This feature and all related services will be deprecated as of August 1, 2021. For more details, see our [New Relic Explorers Hub post](https://discuss.newrelic.com/t/important-upcoming-changes-to-support-and-capabilities-across-browser-node-js-agent-query-builder-net-agent-apm-errors-distributed-tracing/153373).
</Callout>
