We offer a variety of ways to capture distributed tracing data, including:

* [Integrations for third-party telemetry tools](#third-party-integrations)
* [Application agents](#new-relic-agents) (including AWS Lambda and browser monitoring)
* [Our Trace API](#trace-api)

## New Relic integrations for third-party telemetry tools [#third-party-integrations]

If you are collecting telemetry data with selected open source telemetry tools, you can send your data to New Relic with our integrations:

* [Istio](/docs/integrations/open-source-telemetry-integrations/open-source-telemetry-integration-list/new-relics-istio-integration)
* [Kamon](/docs/integrations/open-source-telemetry-integrations/open-source-telemetry-integration-list/kamon-reporter)
* [OpenCensus](/docs/integrations/open-source-telemetry-integrations/open-source-telemetry-integration-list/new-relics-opencensus-integration)
* [OpenTelemetry](/docs/integrations/open-source-telemetry-integrations/opentelemetry/opentelemetry-exporter)

<Callout variant="tip">
  To report your AWS X-Ray telemetry data, see our [AWS integration](/docs/integrations/amazon-integrations/aws-integrations-list/aws-x-ray-monitoring-integration).
</Callout>
