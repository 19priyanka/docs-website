Guided install is a customized CLI command for your environment that downloads and installs the New Relic CLI and the infrastructure agent and detects things to monitor on your system.

Not sure about guided install? Try our add more data page.

## Add more data sources to monitor [#monitor-more]

If you have a [New Relic account](https://newrelic.com/signup), use our [add more data](https://onenr.io/0VRVAy0KMwa) page to browse a catalog of applications, services, and a variety of integrations. 

Not only that, click any tile to get up-to-date how-to installation and configuration steps for whatever you're using.

## Don't have an account? Browse our Instant Observability catalog [#io]

Check our **[Instant Observability quickstarts](https://newrelic.com/instant-observability/)** catalog to get a preview of what you get out-of-the-box. Click **+Install quickstart** to get funneled into the right installation flow.
