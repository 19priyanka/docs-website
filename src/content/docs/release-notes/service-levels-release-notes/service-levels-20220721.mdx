title: 'Limited view for core licence users'
subject: Service levels
releaseDate: '2022-07-21'

**New Relic service levels is now available to core license users (with limitations)**

Core users have now access to SLM in a read only mode. Meaning that they are able to see the list of all the service levels available in their account.

Core users are not able to create, edit or delete service levels, neither opening the service levels details page.
