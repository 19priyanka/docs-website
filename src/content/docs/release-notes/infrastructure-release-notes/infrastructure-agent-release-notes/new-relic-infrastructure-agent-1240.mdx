## Notes
A new version of the agent has been released. Follow standard procedures to [update the Infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).
We recommend you to upgrade the agent every 3 months.

## Added
- https and mTLS support added to the infra-agent HTTP ingest endpoint [#913](https://github.com/newrelic/infrastructure-agent/pull/913)
- Self instrumentation of the agent via Go APM Agent (disabled by default) [#981](https://github.com/newrelic/infrastructure-agent/pull/981)
- Pass NRI_HOST_ID env var into v4 integrations [#1021](https://github.com/newrelic/infrastructure-agent/pull/1021)

## Changed
- Deprecate register_enabled feature flag and add new sdk parameter to prevent entity registration [#1015](https://github.com/newrelic/infrastructure-agent/pull/1015)
