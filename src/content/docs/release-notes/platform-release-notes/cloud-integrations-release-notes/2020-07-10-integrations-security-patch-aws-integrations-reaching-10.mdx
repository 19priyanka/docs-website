### Bug fixes

* As of today we verify that the "External ID" is configured to the [AWS Role](https://docs.newrelic.com/docs/integrations/amazon-integrations/get-started/connect-aws-infrastructure) and we do not accept empty values. This protects customers from cross account attacks.
