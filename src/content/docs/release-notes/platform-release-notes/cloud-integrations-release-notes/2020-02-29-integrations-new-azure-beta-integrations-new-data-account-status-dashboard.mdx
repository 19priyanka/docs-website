### New

* The following Azure integrations have been released as beta:
  * \[Beta] [Azure Machine Learning Services](/docs/integrations/microsoft-azure-integrations/azure-integrations-list/azure-machine-learning-integration)
  * \[Beta] [Azure DataFactory](/docs/integrations/microsoft-azure-integrations/azure-integrations-list/azure-data-factory-integration)

* Account status dashboard now displays the following data:
  * Throttling errors per service
  * Fetching errors
  * Usage of deprecated log ingestion lambda
  * Log-based integration errors
