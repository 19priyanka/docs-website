### New

* The following integrations are now available in beta:
  * [AWS Kinesis Data Analytics](https://docs.newrelic.com/docs/aws-aws_kinesisanalytics-integration)
* New statistics for [AWS ALB](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-albnlb-monitoring-integration) and [AWS ELB](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-elb-classic-monitoring-integration):
  * aws.applicationelb.target_response_time.p90
  * aws.applicationelb.target_response_time.p95
  * aws.applicationelb.target_response_time.p99
  * aws.elb.latency.p90
  * aws.elb.latency.p95
  * aws.elb.latency.p99
* [AWS ElasticCache](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-elasticache-monitoring-integration) Redis Nodes are now decorated with customer tags.
