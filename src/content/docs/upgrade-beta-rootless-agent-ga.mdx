---
title: Upgrade the beta rootless agent to GA
watermark: BETA
---

This document explains how upgrade the rootless agent (`nri-agent`) beta to the GA version.

You need to:

1. Remove all configuration files by uninstalling the beta agent:

   <table>
     <tbody>
       <tr>
         <td style={{ width: "150px" }}>
           **Debian/Ubuntu**
         </td>

         <td>
           ```
           <span style="font-size: 12pt;">dpkg --purge newrelic-infra-rootless-beta</span>
           ```

           or

           ```
           <span style="font-size: 12pt;">sudo apt-get remove --purge newrelic-infra-rootless-beta</span>
           ```
         </td>
       </tr>

       <tr>
         <td>
           **Centos/Suse/RedHat/Amazon**
         </td>

         <td>
           ```
           <span style="font-size: 12pt;">rpm -e newrelic-infra-rootless-beta </span>
           ```

           or

           ```
           <span style="font-size: 12pt;">sudo yum newrelic-infra-rootless-beta </span>
           ```

           or

           ```
           <span style="font-size: 12pt;">sudo zypper rm newrelic-infra-rootless-beta</span>
           ```
         </td>
       </tr>
     </tbody>
   </table>
2. After making sure the beta agent is completely removed, [reinstall the agent selecting the desired running mode](https://docs.newrelic.com/docs/infrastructure/new-relic-infrastructure/installation/install-infrastructure-agent-linux#install-agent).
