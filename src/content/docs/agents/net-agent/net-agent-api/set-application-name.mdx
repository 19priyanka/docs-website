## Syntax

~~~
NewRelic.Api.Agent.NewRelic.SetApplicationName(_string_ $name\[, _string_ $name\_2, _string_ $name\_3\])
~~~
Set the app name for data rollup.

## Requirements

Agent version 5.0.136.0 or higher.

Compatible with all app types.

## Description

Set the application name(s) reported to New Relic. For more information about application naming, see [Name your .NET application](/docs/agents/net-agent/installation-configuration/name-your-net-application).

<div className="callout-important"><p>Updating the app name forces the agent to restart. The agent discards any unreported data associated with previous app names.</p></div>
