### Pass your argument

To pass the `-javaagent` argument on Grails:

<Tabs>
	<TabsBar>
		<TabsBarItem id="grails-run-app">Pass with run-app</TabsBarItem>
		<TabsBarItem id="grails-run-war">Pass with run-war</TabsBarItem>
	</TabsBar>

	<TabsPages>
		<TabsPageItem id="grails-run-app">
            1. Begin with an unzipped version of Grails.
            2. Run this command:

            ```bash
            grails -noreloading -javaagent:<var>/full/path/to/</var>newrelic.jar run-app
            ```
		</TabsPageItem>
		<TabsPageItem id="grails-run-war">
            1. In your Grails app, open this file with your text editor: `grails-app/conf/BuildConfig.groovy`
            2. Add or edit the JVM arguments line:

            ```java
            grails.tomcat.jvmArgs = ["-javaagent:<var>/full/path/to/</var>newrelic.jar"]
            ```
		</TabsPageItem>
	</TabsPages>
</Tabs>
