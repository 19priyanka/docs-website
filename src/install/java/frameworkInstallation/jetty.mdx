### Pass your argument

To pass the `-javaagent` argument on Jetty:

<Tabs>
	<TabsBar>
		<TabsBarItem id="jetty-sh">jetty.sh</TabsBarItem>
		<TabsBarItem id="jetty-startini">start.ini config</TabsBarItem>
	</TabsBar>

	<TabsPages>
		<TabsPage id="jetty-sh">
      If you set exec in `jetty.sh`, edit `JAVA_OPTIONS` in your `jetty.sh` script:

      ```
      export JAVA_OPTIONS="${JAVA_OPTIONS} -javaagent:<var>/full/path/to/</var>newrelic.jar"
      ```

		</TabsPage>
		<TabsPage id="jetty-startini">
      If you set exec in your config file, add the agent path to your `start.ini` config file:

      ```
      -javaagent:<var>/full/path/to/</var>newrelic.jar
      ```      

		</TabsPage>
	</TabsPages>
</Tabs>
