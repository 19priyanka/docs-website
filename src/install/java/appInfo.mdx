**Use guided install to quickly see your data in the UI**

Guided install is a single CLI command you can run on your system to start monitoring your Java app. 

The CLI installs a java-introspector utility attached via our [infrastructure agent](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent). This is a good way to quickly see your Java data on our platform. 

For a more permanent solution and to install the actual Java APM agent, we recommend the manual installation process.
