[New Relic One](/docs/new-relic-one-overview)の基本的なUI機能は、プラットフォーム全体で広く利用できます。こうした機能を理解することで、貴社のデータの発見・分析がさらに容易になります。

## アカウントとエンティティの検索 [#search]

主な検索バーは、すべてのUIページの上部にあります。組織が[複数のNew Relicアカウント](/docs/accounts/accounts-billing/general-account-settings/factors-affecting-access-features-data)にアクセスできる場合、一部のUIページの上部にアカウントセレクターがあります。

* [クロスアカウントアクセス](/docs/accounts/accounts-billing/general-account-settings/factors-affecting-access-features-data#account-access)を使用して、すべてのNew Relicアカウントを検索できます。
* 廃止されるエンティティは、[8日間は検索にヒット](/docs/new-relic-one-entity-explorer#data-retention)します。

## アカウントの調整とユーザーの設定 [#account-dropdown]

アカウント設定とユーザー設定を変更し、サポートとドキュメントリンクを取得するには、ユーザー名の隣の、UIの右上にある[アカウントドロップダウン](/docs/using-new-relic/welcome-new-relic/get-started/glossary#account-dropdown)を使用します。

## New Relic製品に直接アクセスする [#products]

UIの上部には、APM、ブラウザモニタリング、インフラストラクチャモニタリングなど、New Relic Oneの個々の部分への直接リンクがあります。以前の別個の製品から一元化されたNew Relic Oneプラットフォームへの移行の詳細については、[New Relic One移行ガイド](/docs/new-relic-one/use-new-relic-one/core-concepts/new-relic-one-transition-guide-july-2020)をご覧ください。
