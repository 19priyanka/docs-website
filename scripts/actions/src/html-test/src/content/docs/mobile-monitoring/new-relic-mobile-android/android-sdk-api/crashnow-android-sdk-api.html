Syntax

NewRelic.crashNow()
NewRelic.crashNow(string $message)

Throws a demo run-time exception named java.lang.RuntimeException to test New Relic crash reporting.

Requirements

Agent version 4.87.0 or higher.

Description

With this method, you can throw a demo run-time exception named java.lang.RuntimeException with a message. This crash will appear in New Relic Mobile's Crash Analysis page.

Parameters

Parameter

Description

$message

String

Optional. A message attached to the exception.

Examples

Example:

NewRelic.crashNow();

Example with message:

NewRelic.crashNow("This is my test message");
