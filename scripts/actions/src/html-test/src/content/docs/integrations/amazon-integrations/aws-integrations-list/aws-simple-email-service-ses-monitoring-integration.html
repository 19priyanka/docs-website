Amazon Simple Email Service (SES) is a cloud-based service for sending and receiving email. With New Relic's integration with Amazon SES, you can gather metrics and configuration options related to emails sent and received.

Activate integration [#activate]

To enable this integration follow standard procedures to Connect AWS services to New Relic.

Configuration and polling [#polling]

You can change the polling frequency and filter data using configuration options.

Default polling information for the Amazon SES integration:

  • New Relic polling interval: 1 minute resolution

Find and use data [#find-data]

After you connect your AWS integrations, New Relic will start monitoring your AWS data. To find and use your data, including links to your New Relic One dashboards and alert settings: Go to one.newrelic.com > Infrastructure > AWS and select the integration.

You can query and explore your data using these events:

  • SesConfigurationSetSample
  • SesReceiptSample
  • SesRegionSample

Metric data [#metrics]

This New Relic integration collects the following Amazon SES data. For full descriptions of some of these metrics, see Amazon's SNS metrics and dimensions.

Metric

Description

bounce

The number of emails permanently rejected by a recipient's mail server.

click

The number of times the recipient clicked a link in an HTML email.

complaint

The number of times the recipient marked an email as spam.

delivery

The number of emails successfully delivered to the recipient's mail server.

open

The number of times a recipient opened an HTML email.

send

The number of emails attempted to be sent.

Inventory data [#config]

The following configuration options and inventory data are available with the New Relic Amazon SES integration:

aws/ses/region inventory [#region]

Name

Description

accountName

The name of the AWS account.

awsRegion

AWS region of SES endpoint.

maxMailsPerSecond

The maximum number of emails that can be sent per second.

max24HourSend

Maximum number of emails that you can send in a 24-hour period.

aws/ses/receipt-filter inventory [#receipt-filter]

Name

Description

awsRegion

AWS region of SES endpoint.

ipRange

A single IP address or a range of IP addresses to be blocked or allowed.

name

The name of the IP address filter.

policy

Indicates whether to block or allow incoming mail from the specified IP addresses.

aws/ses/receipt-rule-set inventory [#receipt-set]

Name

Description

awsRegion

AWS region of SES endpoint.

name

The name of the receipt rule set.

aws/ses/receipt-rule inventory [#receipt-rule]

Name

Description

actions

List of actions to perform on messages that match the receipt rule.

awsRegion

AWS region of SES endpoint.

enabled

True if the receipt rule is active.

name

The name of the receipt rule.

receiptRuleSetName

The name of the receipt rule set that the receipt rule belongs to.

recipients

The recipient domains and email addresses that the receipt rule applies to.

scanabled

True when messages that this receipt rule applies to are scanned for spam and viruses.

aws/ses/configuration-set inventory [#config-set]

Name

Description

awsRegion

name

The name of the configuration set.

aws/ses/event-destination inventory

Name

Description

awsRegion

AWS region of SES endpoint.

cloudWatchDestinationDimensionsConfiguration

A list of dimensions upon which to categorize emails when email sending events are published to Amazon CloudWatch.

configurationSetName

The name of the configuration set.

name

The name of the event destination.

destination

AWS service to which email sending events are published.

enabled

True when publishing to this destination is enabled.

The type of email sending events published to the event destination.

The ARN of the Amazon Kinesis Firehose stream that email sending events are published to.

snsDestinationTopicArn

Contains the topic ARN associated with an Amazon Simple Notification Service (Amazon SNS) event destination.
