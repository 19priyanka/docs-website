OpenTelemetry provides a single set of APIs, libraries, agents, and collector services to capture distributed traces and metrics from your application. New Relic's OpenTelemetry exporters send your OpenTelemetry data to your New Relic account.

Resources for our OpenTelemetry exporters:

  • Java language exporter
  • Go language exporter
  • .NET language exporter
  • Python language exporter
  • Integration specs
  • For Infinite tracing configuration examples, see Integrations: Enable distributed tracing

Find your data

To find your data, go to one.newrelic.com and go to Your applications > Entity explorer. From the Entities screen, search for your service by name. From there, you can explore your metrics using the Data explorer and build dashboards using your metrics. The distributed tracing feature is also available for you to query and view traces.

For more on how to query your data once it's in New Relic, see Query your data and Introduction to NRQL.
