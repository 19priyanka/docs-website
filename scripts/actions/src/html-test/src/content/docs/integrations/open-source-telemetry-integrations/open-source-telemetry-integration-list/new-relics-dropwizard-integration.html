Dropwizard is a Java framework for developing ops-friendly, high-performance, RESTful web services by providing metrics instrumentation out-of-the-box. New Relic's DropWizard reporter sends your Dropwizard metrics to your New Relic account.

Find your data

To find your data, go to one.newrelic.com and go to Your applications > Entity explorer. From the entities screen, search for your service by name. From there, you can explore your metrics using the Data explorer and build dashboards using your metrics.

For more on how to query your data once it's in New Relic, see Query your data and Introduction to NRQL.
