We offer a cloud integration for reporting your Azure Power BI Dedicated data to our platform. Here we explain how to activate the integration and what data it collects.

Activate the integration [#activate]

To enable the integration follow standard procedures to activate your Azure service.

Configuration and polling [#polling]

You can change the polling frequency and filter data using configuration options.

Default polling information for the Azure Power BI Dedicated integration:

  • New Relic polling interval: 5 minutes

Find and use data [#find-data]

To find your integration data, go to one.newrelic.com > Infrastructure > Azure and select the integration.
