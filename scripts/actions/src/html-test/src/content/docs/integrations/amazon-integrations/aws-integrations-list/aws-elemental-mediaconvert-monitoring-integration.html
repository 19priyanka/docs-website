New Relic includes an integration for reporting your AWS Elemental MediaConvert data to our platform. Here we explain how to activate the integration and what data it collects.

Activate the integration [#activate]

To enable this integration follow standard procedures to Connect AWS services.

Configuration and polling [#polling]

You can change the polling frequency and filter data using configuration options.

Default polling information for the AWS Elemental MediaConvert integration:

  • New Relic polling interval: 5 minutes
  • Amazon CloudWatch data interval: 1 minute

Find and use data [#find-data]

To find your integration data in Infrastructure, go to one.newrelic.com > Infrastructure > AWS and select an integration.
