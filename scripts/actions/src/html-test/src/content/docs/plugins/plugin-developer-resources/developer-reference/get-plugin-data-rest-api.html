Requirements

If you want to use New Relic's REST API (v2) or the API Explorer user interface to get plugin data, you need:

  • Your REST API key
  • The product identification code for your {PLUGIN_ID} or {COMPONENT_ID}
  • Other values as appropriate for your cURL command or API Explorer (v2) request

If you do not know the {PLUGIN_ID} or {COMPONENT_ID}, you can use the REST API (v2) or API Explorer to list each plugin, including its ID, name, GUID, summary metrics, and additional details for each component.
