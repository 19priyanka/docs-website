Notes

New Relic Infrastructure agent builds are available for multiple platforms. See Update the Infrastructure agent for how to download and update the appropriate version of the agent for your platform.

Note also that this is a Windows release.

Bug fixes

  • If you are using the strip_command_line option in its default (true) setting, the command line value will properly handle paths with white space embedded in them.
  • A memory leak found on Windows 10 and Windows Servers 2016 has been fixed.
