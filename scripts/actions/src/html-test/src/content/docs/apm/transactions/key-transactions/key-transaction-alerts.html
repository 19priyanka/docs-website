In APM, you can create alert policies for your key transactions with New Relic Alerts.

Create an alert condition for a key transaction [#create-alert]

To create an alert condition for a key transaction:

  1. Follow the steps to create an alert condition for an APM-monitored entity.
  2. Select Key transaction metric as the type of condition.

View key transaction alert status [#view-alerts]

To view the current health status of key transactions:

  1. Go to one.newrelic.com > More > Key transactions.
  2. View the transactions' color-coded health status.
  3. For more details, hover over the health status.

For more on how to create alert policies and conditions, see New Relic Alerts.
