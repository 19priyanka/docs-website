APM has language-specific agents. All the APM agents have the same general functionality, but there are some agent-specific differences in UI pages and features.

Custom UI pages for APM agents [#agents]

Some agents also have custom UI pages for certain features:

  • C SDK: no C-specific UI pages
  • Go: Go runtime page
  • Java: JVMs page and Instrumentation editor feature
  • .NET: no .NET-specific UI pages
  • Node.js: Node VMs page
  • PHP: WordPress page and Drupal page
  • Python: no Python-specific UI pages
  • Ruby: Ruby VMs page
