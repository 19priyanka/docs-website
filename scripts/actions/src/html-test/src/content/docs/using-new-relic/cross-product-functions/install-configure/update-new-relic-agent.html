Some New Relic solutions require installation of an agent. To ensure that you have the most up-to-date version of an agent, refer to the release notes for that agent. To download and install the latest version, see the docs for that agent.

Update APM agents [#apm]

To see what is the most recent agent version available for your app:

  1. From the account dropdown, select Account settings.
  2. Review the Update your New Relic agent table on the right side of the Summary page.

The UI also includes download links as applicable. For more information, review the docs for your agent:

  • C SDK: Update | Release notes
  • Go: Update | Release notes
  • Java: Update | Release notes
  • .NET: Update | Release notes
  • Node.js: Update | Release notes
  • PHP: Update | Release notes
  • Python: Update | Release notes
  • Ruby: Update | Release notes
