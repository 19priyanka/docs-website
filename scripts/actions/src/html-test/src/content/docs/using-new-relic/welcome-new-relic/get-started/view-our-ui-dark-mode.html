When you're checking on incidents in the middle of the night, or just want to avoid straining your eyes, you can view our UI in dark mode.

To switch between light mode and dark mode in New Relic's UI, click the account dropdown.

Switch between light and dark mode [#dark-mode]

To view our UI in dark mode:

  1. From one.newrelic.com, click the account dropdown. [NOTE: Currently dark mode is not supported in the Firefox browser.]

  2. Click Theme: to switch between the following options:

    • Theme: auto (default): matches the UI with what you already have enabled with your OS.
    • Theme: dark: keeps the UI in dark mode.
    • Theme: light: keeps the UI in light mode.
