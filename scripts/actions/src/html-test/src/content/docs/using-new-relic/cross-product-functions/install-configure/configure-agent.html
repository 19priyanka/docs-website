Some New Relic solutions require installation of an agent. How the behavior of those agents are configured depends on the specific agent.

Configure APM agents [#apm-config]

The only required settings for the APM agents are your license key and app name, which are configured when you install the agent.

For additional configuration options, see:

  • C SDK
  • Go
  • Java
  • .NET
  • Node.js
  • PHP
  • Python
  • Ruby

Configure browser agent [#browser-config]

See Browser monitoring configuration.

Configure infrastructure agent [#infra-config]

The only required setting for the infrastructure agent is your license key, which is configured when you install the agent.

For additional configuration options, see Infrastructure configuration.

Configure mobile agent [#mobile-config]

The only required setting for the mobile agents is the application token, which is configured when you install the agent.

For additional configuration options, see:

  • Android configuration
  • iOS configuration

Configure other New Relic tools [#other]

For configuring other tools not listed here, see the documentation for a specific solution.
