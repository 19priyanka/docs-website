New Relic One has some basic UI functions that are widely available across the platform. Understanding these features will help you find and analyze your data more easily.

Search accounts and entities [#search]

The primary search bar is located at the top of all UI pages. Also, if your organization has access to multiple New Relic accounts, an account selector appears at the top of some UI pages.

  • You can search across all your New Relic accounts with cross-account access.
  • Entities that cease to exist are available in search for 8 days.

Use chart and query features [#data-analysis]

You can add most charts to a new or existing dashboard. Here are some query and chart features available across all or most of the platform:

Do this...

Start querying your data

At the top of any UI page, select Query your data.

View a chart's query

For some charts, you can view the NRQL query used to generate that chart. You can use this to understand a chart better or use it as the basis for a new query.

Choose time range

Drag across a section of a chart to zoom in on that time range. Also, you can use the time picker, which is in the top right corner of the UI, and lets you select pre-set time ranges or set a custom one.

View chart details

Mouse over a chart to see a pop-up with more detail. For some charts, selecting a point on the chart will take you to a UI page with more information about that metric.

Hide or return chart elements

To hide or unhide a displayed chart element, select that element's name below the chart. The chart display will adjust to reflect the absence or presence of that element.

Share charts and pages [#share]

Use any of these options:

Do this...

Embed a chart in a webpage

Click Get chart link to get a URL for embedding a chart in a webpage. An embedded chart contains only the metrics present when it was created. If metrics cease being reported, they won't appear in the chart.

In July 2020, our iframe-based chart-embed feature was updated. For details, see this Explorers Hub post.

Share entire UI page

Copy the URL and share it with anyone who has New Relic access. The URL retains any chart time range settings.

Create a permalink

Click the

icon where available. The permalink also shortens the full New Relic One URL, which makes it easier to share.

Adjust account and user settings [#account-dropdown]

To adjust account settings and user preferences, and to get support and docs links, use the account dropdown, located at the top right of the UI, beside your user name.

Go directly to New Relic products [#products]

At the top of the UI are direct links to individual parts of New Relic One, such as APM, browser monitoring, and infrastructure monitoring. For more about the transition from what were previously separate products to the unified New Relic One platform, see the New Relic One transition guide.
