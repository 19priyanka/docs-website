This doc is for managing users on the New Relic One user model. For managing users on our original user model, see Original users. Learn about user models.

We provide several ways for you to manage your New Relic users. Here are our main user management docs:

  • User model/structure: an explanation of basic aspects of our account/user structure, such as user type (basic user vs full user), default groups (Admin and User), and our standard user roles.
  • Managing users: basics of New Relic user management concepts.
    • A tutorial on how to add new groups and roles
  • User authentication: configure how users are added to New Relic (manually or automatically from an identity provider), and the authentication method (via manual login or SAML SSO).

Your number of full users is a factor in your billing. For more on that, see User count billing calculation.
