In New Relic Insights, there are metric charts available that display metric timeslice data. Because metric charts are a different data type than events, they have different display and customization options.

Metric timeslice data can now be queried using NRQL, giving you more control over how you query and chart this data type.

insights.newrelic.com > Data Explorer > Metrics > (select a dashboard) > Metric charts: To monitor your data in one place, add metric charts to an Insights dashboard via the Metric Explorer in New Relic Insights or via the standard Add to dashboard option in the New Relic UI.

Add metric charts to dashboard [#add-metric-widgets]

Add metric charts to Insights dashboards to:

  • Chart, organize, and monitor metric data in a centralized place
  • Track metrics that the New Relic UI does not chart by default

Add metric charts to an Insights dashboard via the Metric Explorer in New Relic Insights or via the standard Add to dashboard option on any supported New Relic UI metric chart.

You can only chart one metric namespace per chart.

Customize metric charts [#customize]

To customize a metric chart from an Insights dashboard:

  1. From the dashboard where the chart is, select the chart's menu
    icon.
  2. Customize the chart options.
  3. Select Save changes.

Use any of the available options to customize metric charts:

If you want to...

Do this...

View metric values broken out by host

Select Facet by host. This option is available both in the Insights Metric Explorer for the chart at creation time as well as via the metric chart editing UI.

Change the time window

  • Starting time: To change when the chart's time window starts, select the time dropdown.
  • Ending time: All agent metric charts' ending time defaults to now.

Change the title

Titles are automatically generated showing the metric name in many cases. Once an agent metric chart is added to a dashboard, you can edit its title, maximum 255 characters.

Broaden the scope of metrics

To broaden the scope of metrics that are charted, select the metric namespace, and wildcard % any segment of it.

Change the Y-axis value

To change the charted Y-axis value to another value function, select the dropdown. Available value functions depend on the kind of metric data the chart is displaying.

Common examples include:

  • Call count

  • Calls per minute

  • Average requests per minute

  • Maximum

  • Minimum

  • Throughput

    Some charts added via the Add to Insights dashboard chart option may not support changing the value function. To change the value function for the metric, use the Metric Explorer to find and chart it.

Expand the chart

Enlarge the selected chart to get a better view of the data.

Filter on agent metric charts [#filter-metric-widgets]

If your dashboard has filtering enabled, you can still filter on NRQL charts. However, filtering does not apply to any agent metric charts on the dashboard.
