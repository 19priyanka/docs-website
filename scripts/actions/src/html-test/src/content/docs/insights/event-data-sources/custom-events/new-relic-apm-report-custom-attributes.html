Using New Relic APM, you can add your own custom attributes, which you can then use for querying and charting. This is one of several ways to report custom data.

Add custom attributes to APM events [#adding-attributes]

To add custom attributes to APM events, you must first enable them for your New Relic APM agent, and then make an API call to record the attribute. Follow the agent-specific custom attribute procedures:

  • C SDK
  • Go
  • Java
  • .NET
  • Node.js
  • PHP
  • Python
  • Ruby

Requirements for custom attributes [#requirements]

When creating your own custom attributes, follow data requirements for:

  • Size limits
  • Attribute types
  • Reserved words
