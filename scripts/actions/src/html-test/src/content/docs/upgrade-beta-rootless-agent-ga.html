This document explains how upgrade the rootless agent (nri-agent) beta to the GA version.

You need to:

  1. Remove all configuration files by uninstalling the beta agent:

    Debian/Ubuntu

    dpkg --purge newrelic-infra-rootless-beta


    
sudo apt-get remove --purge newrelic-infra-rootless-beta
    or

    Centos/Suse/RedHat/Amazon

    rpm -e newrelic-infra-rootless-beta


    
sudo zypper rm newrelic-infra-rootless-beta
    or 
    
sudo yum newrelic-infra-rootless-beta
    or

  2. After making sure the beta agent is completely removed, reinstall the agent selecting the desired running mode.
