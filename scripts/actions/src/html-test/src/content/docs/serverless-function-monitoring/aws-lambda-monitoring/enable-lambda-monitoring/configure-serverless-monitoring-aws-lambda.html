After you set up serverless monitoring for AWS Lambda, you can add additional configuration to fine-tune your data.

Set up alerts [#alerts]

Our alerts let you get notifications on anything you can monitor with New Relic, including APM metrics, key transactions, NRQL queries, and more.

For details about setting up alerts on Lambda functions, see monitoring for AWS Lambda: Configuring alerts

Add custom events [#custom-events]

Besides the data we provide by default, you can also set up your own events or attributes.

For details about these language-specific settings, see Configuring custom attributes and events in AWS Lambda.
