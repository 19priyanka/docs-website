This document explains how to uninstall the New Relic Java agent. For instructions on how to temporarily disable the agent, see Disable the agent.

To remove the Java agent:

  1. Delete the newrelic folder from your app server's root directory.
  2. Remove the JVM argument you used to include the agent in your startup script.
  3. If applicable, rename your backed-up start script to its original name.
  4. After you complete these steps, restart your app.

You may also want to remove your app from the APM UI.
