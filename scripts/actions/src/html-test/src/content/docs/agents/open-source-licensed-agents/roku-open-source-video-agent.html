Monitor Roku behavior with New Relic using the Roku open-source agent. The agent contains two parts, to capture two separate categories of Roku behavior:

  1. App events like app starts and HTTP requests
  2. Video playback within the app

This agent is released as open source on GitHub. A change log is also available there for the latest updates.
