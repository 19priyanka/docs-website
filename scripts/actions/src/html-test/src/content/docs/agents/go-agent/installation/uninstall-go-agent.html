Read on to learn how to uninstall the New Relic Go agent. For instructions on how to temporarily disable the Go agent, see Disable the agent.

To uninstall New Relic for Go:

  1. Uninstall the New Relic package using your preferred uninstall method.
  2. Recompile and restart your application.

Once you uninstall the New Relic package, you will be able to remove the application from your New Relic account.
