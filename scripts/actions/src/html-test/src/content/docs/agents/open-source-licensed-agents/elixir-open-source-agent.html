Monitor Elixir behavior with New Relic using the Elixir open-source agent. The agent:

  • Helps you track transactions, distributed traces, and other parts of your application’s behavior
  • Provides an overview of underlying BEAM activity

This agent is released as open source on GitHub. A change log is also available there for the latest updates.

Get started [#get_started]

For requirements, installation, and configuration information, see the Open Source Elixir Agent README on GitHub.

Visit New Relic’s Elixir repository on GitHub for questions about installation, usage, or other topics. Report issues or bugs as an issue in the GitHub repository.
